# 文件上传与标注应用

## How to Run

### 使用 Docker Compose（推荐）

```bash
# 构建并启动所有服务
docker-compose up --build -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

### 本地开发

```bash
# 进入前端用户端目录
cd frontend-user

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 项目介绍

这是一个基于 Next.js 14 和 Tailwind CSS 开发的文件上传与标注应用，支持 PDF 和图片文件的上传、预览和标注功能。

### 技术栈

- **框架**: Next.js 14.2.21
- **UI**: Tailwind CSS 3.4.17
- **语言**: TypeScript 5.7.2
- **PDF 渲染**: pdfjs-dist 4.9.155

### 功能特性

#### 文件上传
- 支持拖拽上传和点击选择
- 支持 PDF、JPG、PNG、GIF、WebP 格式
- 文件大小限制 50MB
- 上传进度显示

#### 标注工具
- 🖊️ 画笔 - 自由绘制
- 🖍️ 荧光笔 - 半透明高亮
- 📝 文字 - 添加文字标注
- ⬜ 矩形 - 绘制矩形框
- ⭕ 圆形 - 绘制椭圆
- ➡️ 箭头 - 绘制箭头
- 🔍 选择 - 选中并移动标注
- 🧹 橡皮擦 - 删除标注

#### 其他功能
- 颜色选择（8 种预设颜色）
- 线宽调节（1-20）
- 撤销/重做
- 清除所有标注
- 导出标注数据（JSON 格式）
- PDF 分页浏览和缩放
